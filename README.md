# Governed AI Architecture

This repository contains a governed AI architecture designed to prioritise:

- human agency
- epistemic restraint
- silence as a first-class operation
- long-timescale coherence

This is **not** a prompt pack.
This is **not** a productivity system.
This is **not** a replacement for judgment.

Read the constitution first.

## Non-Goals

This repository is not intended to:
- maximise productivity
- provide therapeutic advice
- automate decision-making
- replace human judgment
- serve as a generic AI prompt framework

If those are your goals, this architecture is not suitable.
